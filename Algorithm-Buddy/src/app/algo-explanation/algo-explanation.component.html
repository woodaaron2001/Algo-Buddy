Note this site is designed for people with knowledge of speedcubing however if you are new and need some information about solving:

First how do we solve a rubiks cube?

First we need to know what the most popular method of solving a cube is: CFOP!

CFOP breaks solving the cube into layers rather than solving each face. It has 4 seperate stages:

cross

F2L

OLL

PLL


The cross

The cross is the first step which requires creating a cross where each edge of the cross aligns with the colour on the adjacent center. This however can be done intuitvely so is not included in the list of algorithms


F2L: First 2 Layers

The second step is F2L here we begin to match corner-edge pairs and insert them into the slots between the cross to build up the first 2 layers

This can be done intuitvely however fast solvers use sequences of moves called algorithms to spot a case and instantly solve it by memorization


"Cases"

A Case can be deemed as a recognizable set of edges and corners which can be solved by some predefined set of moves.
